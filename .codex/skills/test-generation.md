# Skill: test-generation

## Goal

Generate Salesforce test scenarios and implementation test plans.

## Scenario types

- Positive scenarios
- Negative scenarios
- Edge cases
- Null or missing values
- Bulk processing
- Permission scenarios
- UI validation scenarios

## Apex test expectations

- Tests assert business outcomes.
- Tests do not exist only for coverage.
- Test data is clear and isolated.
- Bulk behavior is covered when relevant.
- Error and validation behavior is covered.

## Output

- Test matrix
- Apex tests to create/update
- Manual validation steps
- UAT scenarios
- Remaining test gaps
