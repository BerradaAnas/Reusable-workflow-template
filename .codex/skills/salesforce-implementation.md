# Skill: salesforce-implementation

## Goal

Implement only approved Salesforce work safely in a local Salesforce DX project.

## Rules

- Work only after explicit ACT MODE approval.
- Modify only approved components/files.
- Respect existing project conventions.
- Keep changes minimal and reviewable.
- Do not deploy by default.
- Do not perform destructive changes.
- Add or update tests when code changes.

## Output after work

- Files changed
- Reason for each change
- Tests added/updated
- Checks to run
- Known limitations
