# Skill: lwc-review

## Goal

Review Lightning Web Components for usability, reliability, accessibility, and Salesforce integration quality.

## Checklist

- Accessible labels and semantic markup
- Loading states
- Friendly error messages
- Empty-state handling
- Permission-aware UI behavior
- Safe Apex integration
- Reactive state handling
- Jest test readiness
- Clear component API

## Output

Group findings by severity:

- Critical
- High
- Medium
- Low

For each finding, include:

- Issue
- Why it matters
- Suggested fix
