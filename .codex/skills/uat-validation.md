# Skill: uat-validation

## Goal

Convert acceptance criteria into clear UAT scripts for business validation.

## Include

- Scenario name
- Preconditions
- Test data
- Step-by-step actions
- Expected result
- Actual result
- Pass/fail
- Defect notes

## Rules

- Link each scenario to acceptance criteria.
- Keep language business-friendly.
- Include negative and edge scenarios where useful.
- Separate setup steps from validation steps.

## Output

- UAT test script
- Test data needs
- Pass/fail criteria
- Defect capture notes
