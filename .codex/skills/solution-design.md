# Skill: solution-design

## Goal

Produce a safe Salesforce implementation plan before coding.

## Include

1. Proposed design
2. Data model impact
3. Automation impact
4. Apex impact
5. LWC/UI impact
6. Security, CRUD, FLS, and sharing impact
7. Test strategy
8. Deployment plan
9. Rollback plan
10. Risks and assumptions

## Rules

- Prefer minimal, reviewable changes.
- Explain tradeoffs.
- Avoid overengineering for MVP tickets.
- Identify human approval gates clearly.
